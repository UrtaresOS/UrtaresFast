# UrtaresFast

Bilgisayarınızdaki tüm gereksiz dosyaları temizler ve cihazınızı hızlandırır!

## Nasıl indirilir?

İndirme simgesine tıklayın.
Download ZIP'e basın.
Gelen dosyayı açın ve urtaresfast.bat dosyasını açın.
Bitmesini bekleyin.
